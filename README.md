# 🏠 Rental App - Mobile UI/UX Design

## 📌 Overview
This project is a **mobile application UI/UX design for a Rental App**, created using Figma.  
The goal is to design a clean, modern, and user-friendly experience for browsing and renting properties.

---

## 🎯 Purpose
- Practice UI/UX design skills  
- Design a real-world rental platform flow  
- Improve mobile app design thinking (user experience & usability)

---

## 🎨 Design Tool
- Figma

## 🔗 Figma Design Link
https://www.figma.com/design/bUrooN7b7pBsaseAvjLPD0/rentalapp

---

## 📱 App Features (UI Design)
- 🏠 Home screen with property listings  
- 🔍 Search & filter properties  
- 📄 Property detail page  
- 📅 Booking / rental flow  
- 👤 User profile screen
